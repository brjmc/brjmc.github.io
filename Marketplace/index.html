The Marketplace is a full stack web application built for a computer engineering course at UBC. You are looking at a simplified version that receives randomized product information from a simple server. The actual, complete version (the code for which lives here), makes POST and GET requests to a more complex backend that authorizes requests and stores product information in a MongoDB database.
